# Тестовое задание на позицию Go - developer

## Настройка окружения
В корневой папке в файле .env содержатся все переменные окружения, которые нужны для конфигурации приложения и его работы.
<b>Ссылка на внешнее API присваевается без trailing слеша, то есть без '/' в конце ссылки</b>

Логгирование включает в себя три режима:
- local (текстовый формат на уровне DEBUG, значение по умолчанию)
- dev (JSON формат на уровне DEBUG)
- prod (JSON формат на уровне INFO)

<b>Перед запуском следует настроить базу данных</b>

## Запуск приложения
Запустить приложение можно двумя способами
```
go run cmd/cars/main.go
```
или
```
make up
```
